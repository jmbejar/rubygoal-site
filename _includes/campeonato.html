Para participar en el campeonato de fútbol RubyGoal hay que implementar una clase que herede de Coach e implemente los métodos name y formation.
{% highlight ruby %} module Rubygoal class MyCoach < Coach def name "My team" end def formation(match) formation = Formation.new formation.defenders = [:average, :average, :none, :average, :average] formation.midfielders = [:average, :none, :fast, :none, :average] formation.attackers = [:none, :fast, :captain, :fast, :none ] formation end end end {% endhighlight %} El jugador cuenta con la siguiente información del partido para ir adaptando su táctica a lo largo del partido. Esta información es accesible desde el parámetro match que recibe el método formation.

Match#time
indica cuanto tiempo (segundos) le queda al partido.
Devuelve un número entre 120 y 0 (120 cuando comienza el partido) {% highlight ruby %} if match.time < 10 formation.defenders = [:none, :none, :none, :none, :none] end {% endhighlight %}

Match#me
devuelve un objeto con la siguiente información
draw? Devuelve true si el partido está empatado
losing? Devuelve true si tu equipo va perdiendo
winning? Devuelve true si tu equipo va ganando
score Devuelve la cantidad de goles que ha hecho tu equipo
{% highlight ruby %} if formation.me.winning? || formation.me.draw? formation.defenders = [:average, :fast, :captain, :fast, :average] elsif formation.me.losing? formation.attackers = [:average, :fast, :captain, :fast, :average] end {% endhighlight %}

Match#other
devuelve un objeto con la siguiente información
formation Devuelve la alineación del equipo rival
draw? Devuelve true si el partido está empatado
losing? Devuelve true si el equipo rival va perdiendo
winning? Devuelve true si el equipo rival va ganando
score Devuelve la cantidad de goles que ha hecho el equipo rival
{% highlight ruby %} rival = formation.other.formation if rival.attackers.include?(:captain) formation.defenders = [:average, :fast, :captain, :fast, :average] end {% endhighlight %}
